BucketList

iOS Mobile Application

BucketList was created in 2019/2020 with the goal of helping random folks track what they want to achieve in life.

Whatever you want to do, note it down in BucketList.

BucketList is created with a simple design to ensure that users find it easy to just pen down what they want to achieve.

Bucketlist is created with XCode using Swift.
