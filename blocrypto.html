Blocrypto

Web3 Platform

Blocryptor is a full-stack application designed to help students and developers in the Web3 space analyze smart contracts for vulnerabilities. With a React frontend and a Flask backend, it integrates with APIs such as OpenAI and blockchain explorers like Etherscan, BscScan, and PolygonScan. As a trusted third-party tool, Blocryptor provides reliable insights, making it easier to verify and implement secure smart contracts.

Key Features

  • Smart Contract Analysis
  • Blockchain Explorer Integration
  • AI-Powered Insights
  • User-Friendly Interface
  • Secure & Scalable Backend
  • Third-Party Trust & Verification

Technologies Used

  • React.js
  • Flask (Python)
  • OpenAI, Etherscan, BscScan, PolygonScan
  • Ethereum, Binance Smart Chain (BSC), Polygon
