Back in 2017, a friend and I felt that social media was rigid. It didn't allow us to post at multiple platforms at the same time

We decided to take action in creating a platform that allows users to post at multiple platforms in one go. It was called SMTool.

It was my first time interacting with APIs of different platforms and seeing it works. Going through each authentication and authorisation process.

SMTool was developed using PHP with the mySQL Database in used.
